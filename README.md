# Lloga — Landing page

Landing page premium para **Lloga**, marca de yoga contemporánea en Köln. Página única y autocontenida (`index.html`): HTML + CSS + JS + GSAP + tipografía inline, sin dependencias externas.

## Ver en vivo

Activa GitHub Pages en este repo (Settings → Pages → Branch: `main` → `/root`) para tener una URL pública, o abre `index.html` directamente en el navegador.

## Contenido

- Copy en alemán, orientado a principiantes en Köln.
- Paleta stone-greige cálida con salvia y azul-gris como acentos contenidos en tarjetas.
- Tipografía Fraunces (display) + Karla (cuerpo), autohospedada.
- Motion con GSAP + ScrollTrigger (inline).
- Modo claro/oscuro (`prefers-color-scheme` + toggle manual vía `data-theme`).

## Pendiente antes de publicar en producción

- Dirección real del estudio, horarios, precios.
- Testimonios reales (los slots actuales son placeholders marcados).
- Cuenta de WhatsApp Business e Instagram reales.
- Biografía y credenciales reales de la profesora.
