# Lloga Landing page

Landing page premium para **Lloga**, marca de yoga contemporánea en Köln. Página única y autocontenida (`index.html`): HTML + CSS + JS + GSAP + tipografía inline, sin dependencias externas.

## Ver en vivo

Activa GitHub Pages en este repo desde Settings, Pages, Branch `main`, carpeta `/root`, o abre `index.html` directamente en el navegador.

## Contenido

- Copy en alemán, orientado a principiantes en Köln.
- Paleta stone greige cálida con salvia y azul gris como acentos contenidos en tarjetas.
- Tipografía Fraunces para display y Karla para cuerpo, autohospedada.
- Motion con GSAP y ScrollTrigger inline.
- Modo claro y oscuro con `prefers-color-scheme` y toggle manual vía `data-theme`.
